Project 3: COVID-19 Outcomes Regression

Forecasting outcomes among vaccinated people (per 100k) using rates, populations, and age

Introduction

🎯 Motivation

Predict the expected rate of outcomes among vaccinated people (per 100,000) to support planning and communication.

This project analyzes structured records containing vaccination rates (unvaccinated/vaccinated/boosted), population counts, age groups, and health outcomes to build predictive models for public health planning.

📊 View Source Dataset

Problem Statement & Research Question

🎯 Problem

Build a regression model to predict Outcome_Vax_per100k.

❓ Research Question

How well do vaccination rates, population scale, and age explain vaccinated outcome rates, and which linear specification performs best?

Data Understanding

📈 Univariate

Heavy right-skew in rates and outcomes; multimodal populations.

🔄 Transforms

Applied log1p to stabilize variance.

📊 Bivariate

Strong positive association with Unvaccinated Rate; near-deterministic tie spotted.

👥 Groupwise (Age)

Higher spread in older groups; interactions plausible.

🔗 Multicollinearity

VIF flagged Vaccinated vs Unvaccinated Rates (>6).

📋 Long-format

Reshape to compare Unvax/Vax/Boost rate–outcome pairs consistently.

⚠️

Data Leakage Prevention

I avoid leakage by excluding Vaccinated Rate when modeling vaccinated outcomes directly tied to its denominator.

Preprocessing Pipeline

🚫 Remove leakage features
🧩 Median/mode imputation
🔁 log1p transform on skewed numerics
🔤 One-hot encode Age
📏 Standardize numerics
🔀 Train/test split

Pipelines ensure no data leakage and reproducible transforms.

Model 1 — Required Linear Regression

🎯 Chosen Specification (A2)

OLS, no interactions, keep populations

All-Subsets Search

I built a comprehensive search function that systematically evaluates model performance across multiple dimensions:

🔍 Feature Sets

  • • rates_only
  • • rates + pop_vax
  • • rates + pop_unvax
  • • rates + pop_boost

🤖 Models Tested

  • • OLS Regression
  • • Huber Regression
  • • ElasticNetCV
  • • Gradient Boosting (GBR)

Each combination is fitted on the training set using our preprocessing pipeline (impute → log1p → scale → one-hot), then evaluated on the test set with back-transformation to original scale for interpretable metrics.

Key Findings

🎯

Best Linear Model

Simple OLS without interactions achieved the best practical accuracy with MAE of 18.67 per 100k.

⚡

Interaction Effects

Removing polynomial interactions significantly reduced overfitting in tail predictions.

This regression analysis provides a robust foundation for predicting COVID-19 outcomes among vaccinated populations, supporting evidence-based public health decision making.
